# Tijhuis-Tuinen  
Website Project for a Landscaping Company

## Overview
[Tijhuis-Tuinen.nl](https://tijhuis-tuinen.nl/) is a responsive, modern website built using Bootstrap 5, Swiper sliders, and Bootstrap Icons.  
The repository serves as the testing environment for the production website of the landscaping company, where I manage all IT-related tasks.

### Testing Environment
[https://stensel8.github.io/Tijhuis-Tuinen/](https://stensel8.github.io/Tijhuis-Tuinen/)

### Production Website
[https://tijhuis-tuinen.nl/](https://tijhuis-tuinen.nl/)

---

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/Stensel8/Tijhuis-Tuinen.git
   ```
2. Open the `index.html` file in your browser  
   or deploy the project via GitHub Pages.
